# v0.1.5: Practical Toolkit Pass

This release turns OSS Supporter from a playbook into a small reusable toolkit.

## Added

- Agent-first entrypoint: `AGENT_START.md`.
- Minimal private workbench example.
- Full lifecycle case study.
- Portable helper tools:
  - lead scoring
  - repo capability preflight
  - generated/source-truth check
  - PR body builder
  - cleanup doctor
  - public boundary checker
- Public helper flow visual.
- Lightweight Python check runner.
- CI for tool tests and public-boundary validation.
- Refreshed May 2026 impact receipt.

## Still Intentional

- No automated public comments.
- No raw transcript publishing.
- No private queue exports.
- No vendor lock-in requirement.

## Upgrade Path

Read `AGENT_START.md`, then use `docs/workflow-upgrades.md` for tool flow.
