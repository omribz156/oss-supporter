# OSS Supporter v0.1.0

Initial public release candidate.

## Included

- Public playbook for careful agent-assisted OSS support.
- Validation gates for avoiding stale, duplicate, or noisy contribution targets.
- Public voice guidance for transparent, maintainer-first comments and PRs.
- Review follow-up and cleanup guidance.
- Work-slice, lead-claim, PR body, issue comment, status, and impact templates.
- Privacy-first token impact meter for Codex JSONL logs.
- First public impact snapshot for May 2026.
- Public presentation assets and README visuals.

## Boundary

This release contains methods, templates, sanitized examples, and aggregate
impact. It does not include private workbench logs, raw sessions, credentials,
account/legal state, or hidden queues.
