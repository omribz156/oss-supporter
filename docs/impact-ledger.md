# Impact Ledger

Impact should be public, aggregate, and honest.

## Track

- issues triaged
- PRs opened
- PRs merged
- review fixes made
- projects helped
- maintainers unblocked
- estimated tokens used for OSS support
- estimated cost, when calculable

## Do Not Track Publicly

- raw prompts
- raw logs
- private session IDs
- account IDs
- hidden queues
- private maintainer contact
