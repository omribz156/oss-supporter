# Review Follow-Up

Maintainer comments are the highest-signal learning source.

## Status Check

For each open public action, check:

- state: open, merged, closed, draft
- maintainer comments
- reviews and inline comments
- CI failures
- legal/CLA gates
- mergeability
- duplicate or superseding PRs

## Decision Words

- clear: no requested action; wait
- fix: actionable feedback; patch if small
- blocked-user: legal/account/private action needed
- blocked-maintainer: maintainer decision or permission needed
- merged: record and clean up
- closed/dropped: record outcome and clean up
- unclear: fetch one more specific source, then decide

## Lessons

When feedback teaches a reusable rule, record:

- date
- project
- signal
- lesson
- future action pattern

Do not repeat the same mistake across projects.
